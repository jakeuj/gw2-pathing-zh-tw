FAQ

常見問題

現在可以下載繁體中文標記包嗎？

還不行。這個 repository 目前是授權與風險管理紀錄，不存放或散布未授權的標記包內容。日後只有通過發布條件的標記包才會各自發布。

為什麼非商業翻譯仍然需要許可？

因為翻譯涉及改作，而可運作的獨立套件也可能需要再散布原始檔案。非商業並不自動取代授權或權利人的明確許可。

公開在 GitHub 上不就代表可以修改嗎？

不代表。repository 公開、沒有回覆，或回覆內容模糊，都不視為允許改作與再散布。

授權完成後會一次發布全部內容嗎？

不會。每個標記包會分別確認條件、釘選上游版本、翻譯、署名並發布，避免一個包的狀態影響其他包。

這是 ArenaNet、Blish HUD 或 Pathing 的官方網站嗎？

不是。這是社群維護的非官方在地化計畫，與 ArenaNet、Blish HUD、Pathing 及各標記包作者沒有官方從屬關係。
